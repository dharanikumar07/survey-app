  • has_action( $hook ) only returns true if something is already hooked into that action.

  • But in WooCommerce, the hooks like woocommerce_single_product_summary may not have anything hooked yet when your function runs.

  • That means has_action( 'woocommerce_single_product_summary' ) might return false, so your add_action() never runs.

👉 In most cases, you don’t need to wrap add_action() inside a has_action() check. Just add it directly.
